
### 1.Cookie

脚本顶部有个 `COOKIE` 变量，换成你自己的微博 Cookie。怎么拿：

1. 浏览器登录微博
2. F12 打开开发者工具 → Network
3. 随便刷一下页面，点任意一个请求
4. 把 Request Headers 里的 `Cookie` 整行复制过来

Cookie 会过期，抓着抓着报 403 就是该换了。

### 2. 改目标用户

顶部 `TARGET_UID` 改成你想抓的用户的 UID。

### 3. 跑

```bash
python3 weibo_scraper.py
```

输出在 `weibo_data/` 目录下，三个文件：

- `weibo_{UID}.jsonl` — 结构化数据，一行一条微博，方便程序处理
- `weibo_{UID}.txt` — 纯文本，方便人看
- `weibo_{UID}_meta.json` — 抓取汇总信息


## 配置项


| 变量 | 说明 | 默认值 |
|------|------|--------|
| `TARGET_UID` | 目标用户 UID | - |
| `OUTPUT_DIR` | 输出目录 | `weibo_data` |
| `COOKIE` | 微博 Cookie | - |
| `SKIP_COMMENTS` | 跳过评论抓取 | `False` |
| `SKIP_LONGTEXT` | 跳过长微博展开 | `False` |
| `PAGE_SIZE` | 每页微博数 | `20` |
| `MAX_CMT_PAGES` | 单条微博最多抓几页评论 | `30` |
| `DELAY` | 翻页间隔（秒），随机范围 | `(3, 6)` |
| `CMT_DELAY` | 评论翻页间隔 | `(2, 4)` |
